# Node Client

Example placeholder for a Node.js client.

## Install

```bash
npm install @reflog/client
```

## Usage

```js
import { ReflogClient } from "@reflog/client";

const client = new ReflogClient({ url: "http://localhost:50051" });
await client.ingest({
  entity: "user_123",
  op: "UPDATE",
  payload: { status: "active" },
});
```
